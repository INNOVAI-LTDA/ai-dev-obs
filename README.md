# AIDevObs

AIDevObs is an observability platform for AI-assisted software development.

It helps developers understand how AI is being used during software creation by tracking:

- Sessions
- Prompts
- Responses
- Context usage
- Estimated token consumption
- Development time
- Explainability artifacts
- Productivity insights

AIDevObs is designed for developers, architects, consultants and teams adopting AI-assisted software engineering practices.
